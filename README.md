# 📱 TaskFlow

O TaskFlow é um aplicativo Android de gerenciamento de tarefas desenvolvido com **Kotlin** e **Jetpack Compose**.
O objetivo do projeto é oferecer uma forma simples e eficiente de organizar tarefas do dia a dia.

---

## 🚀 Funcionalidades

* Criar, editar e excluir tarefas
* Filtrar tarefas (Todas, Pendentes, Concluídas)
* Acompanhar o status de cada tarefa
* Interface moderna e intuitiva

---

## 🛠️ Tecnologias utilizadas

* Kotlin
* Jetpack Compose
* Arquitetura MVVM
* Android Studio

---

## 📂 Estrutura do projeto

* `ui/` → Interface e componentes visuais
* `viewmodel/` → Lógica de negócio
* `model/` → Estrutura dos dados

---

## 🎯 Objetivo

Este projeto foi desenvolvido com foco no aprendizado de desenvolvimento Android moderno, utilizando boas práticas e arquitetura limpa.
